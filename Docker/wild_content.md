# Docker

## Définition rapide

Docker est une plateforme open source qui permet de créer, déployer et exécuter des applications dans des conteneurs légers et portables. Un conteneur regroupe une application et ses dépendances, assurant une exécution cohérente sur différents environnements.

## Mots-clés

Conteneurisation, Virtualisation légère, Images, Isolation, Portabilité, Volumes, Nextwork, Port
